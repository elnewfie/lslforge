Examples

Examples are provided to illustrate some of the features of LSL Plus. Currently, three samples are provided, each of which illustrate a 'Sim Project' in conjunction with and event handler module:

  • HTTP/RPC Example: illustrates a script that uses script-initiated HTTP requests (llHTTPRequest) and XML-RPC, along with an event handler module to simulate incoming XML-RPC requests.
  • Dialog Example: illustrates a script that scans for a nearby avatar, and presents a dialog to that avatar, along with an event handler module to simulate the avatar making a selection from the dialog.
  • HTTP Request Example: illustrates a stript that registers an URL for use with the 'http_request' event, and a handler module that initiates incoming HTTP requests.

The examples can be installed into an LSL Plus project by selecting (from the context menu within the Navigator view, from File -> New) the appropriate example creation wizard:

You select the location where you want it installed, and can then look at the source code and run the sample.
